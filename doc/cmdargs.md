## rssforward.py --help
```
usage: rssforward.py [-h] [-c CONFIG] [--no-tray] [--no-server]

RSS Forward

optional arguments:
  -h, --help            show this help message and exit
  -c CONFIG, --config CONFIG
                        Path to TOML config file
  --no-tray             Do not use system tray icon (overrides config
                        'trayicon' option)
  --no-server           Do not run RSS server (overrides config 'startserver'
                        option)
```
